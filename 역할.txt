이혜연 : Redux 슬라이스 및 액션 생성자 작성
상태 관리와 연산을 처리하는 부분 및 createSlice를 사용하여 Redux 슬라이스를 생성
초기 상태와 리듀서 함수를 정의
더하기, 빼기, 곱하기, 나누기, 초기화 등 다양한 연산을 수행하는 액션 생성자를 작성

변형준 : 스토어 설정 작성
Redux 스토어를 설정하고, 애플리케이션의 상태를 관리,
configureStore를 사용하여 Redux 스토어를 생성 및
애플리케이션의 여러 리듀서 중에서 계산기 리듀서를 스토어에 등록합니다.

강은구 : 컴포넌트 및 사용자 입력 처리 
사용자 인터페이스를 구성하고 사용자 입력을 처리하여 Redux 액션을 디스패치
React 함수 컴포넌트를 생성 및 useState를 사용하여 사용자의 입력을 관리하는 상태를 생성
useDispatch와 useSelector를 사용하여 Redux 스토어와 연결
